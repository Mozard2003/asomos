使用條款 & 使用者隱私

條款版本: 第 1.4 版

更新日期: 2024 / 5 / 30

免責聲明

1. 開始使用本專案代表你已經同意我們的使用條款。 因此在開始使用本專案 (即邀請機器人至群組，或是使用機器人之指令) 之前，請閱覽我們的使用條款，保護自身權益。 如果對本條款有任何疑問，可以詢問 Asomos 的開發團隊。

2. 我們不保證本專案能絕對安全、無錯誤的運行，請使用者自行承擔使用本專案之風險。

3. 我們不會承擔任何使用者失誤所導致的後果。

第一部分: 關於我們

1. 我們是 Asomos 機器人的開發團隊。 如果有需要可以加入我們的官方社群，與我們聯絡。

2. Asomos 機器人，主要提供與 Discord 使用者進行互動的服務。

第二部分: 使用規範

1. 我們要求所有使用本專案之使用者必須遵守 Discord 的服務條款 & 社群守則。

2. 本專案會嘗試回應每個下達的指令，但是禁止透過本專案來洗版、惡意濫發訊息、破壞其他使用者的使用體驗、增加系統或 API 的負擔等行為。

3. 使用者使用以下指令: choose、clear ，所生成的訊息內容或結果負有完全責任，請確保生成之內容或結果不會造成他人困擾或是違反相關規定。

4. 使用者嘗試或利用本專案觸犯包含但不限於使用者所屬群組的規定、 Discord 的服務條款、社群守則或使用者所屬地區的法規， Asomos 機器人及其開發團隊成員並不會承擔任何責任。

5. 若有不正當的使用行為，我們有權力停止服務特定的使用者或群組。

第三部分: 使用者隱私

1. 部分指令會透過 API 取得 Discord 使用者、群組、頻道、身分、表情符號、貼圖、頭像或是訊息的資料作為指令的回應。

2. 我們不會隨意竊取使用者的對話紀錄和內容，但指令觸發時，我們會紀錄指令的使用狀況，做為開發用的統計或除錯，以下為紀錄內容:

  • 指令名稱
  • 指令類型
  • 輸入內容
  • 使用者名稱與 ID
  • 群組名稱與 ID
  • 觸發時間
  • 終端是否報錯

3. 開發團隊可以查看機器人所在群組的以下資訊:

  • 群組名稱
  • 群組 ID
  • 群組人數
  • 群組使用者
  • 群組擁有者

4. 我們會保護以上資料，除非涉及法律或違反相關規範，一般情況下不會將它提供給任何非本團隊的人。

4.1. 使用者在有正當理由的情況下，可以向我們申請查看指令使用紀錄，但查看範圍僅限該群組成員的使用紀錄，且理由須經團隊成員審核通過。

5. 使用者若是使用指令: report ，部分的回報的內容是會公開顯示於我們社群的頻道，我們將會公開以下資訊:

  • 回報類型
  • 回報內容
  • 回報者名稱與 ID
  • 回報群組名稱與 ID
  • 回報時間

5.1. 使用 report 指令後，我們可能會利用以下三種方式進行回覆:

  • 私人訊息
  • 指令觸發的頻道
  • 開發者的社群

第四部分: 智慧財產權

1. 本專案包含但不限於機器人帳號、官方社群、程式碼、企業識別、本網站等開發團隊用於開發或運行機器人之設計，除了部分內容。 我們對這些內容保有一切智慧財產權。

2. 禁止未經過同意將本專案用於任何商業用途。

第五部分: 法律 & 爭議

1. 本專案與條款，依照中華民國(臺灣)法律為準。

2. 有任何爭議發生時，希望雙方能以和解的方式處理。 若無法達成和解，雙方經過同意後，將爭議提交至具有管轄權之法院。 若使用者並非中華民國人，請參考中華民國民事訴訟法第 402 條。

第六部分: 服務的更動

1. 我們具有隨時變更或終止本專案運作或服務的權力。

2. 本條款會因應需求而隨時進行修正。 建議定期查看使用條款，保護自身權益。

3. 使用者有權利隨時無條件停止使用本專案提供的服務。
